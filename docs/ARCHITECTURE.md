# MiHub Personal - Architecture

## Overview
Personal task management application built with Vue 3 and Spring Boot.

## Features
- Add tasks with title, priority, and optional due date
- Automatic sorting by urgency (due date first, then priority)
- Complete/uncomplete tasks
- Delete tasks
- LocalStorage persistence

## Tech Stack
- Frontend: Vue 3, TypeScript, Tailwind CSS, Pinia
- Backend: Spring Boot 3.3, Java 21, JPA
